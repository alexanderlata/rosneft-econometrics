1 Предельные значения

1.1 для probit

Рассмотрим probit-модель \(P(y=1)=\Phi(x'\beta)\), где \(x'\beta=\beta_0+\beta_1x_1+\dots+\beta_kx_k\)

Предельные значения \(\frac{\partial P(y=1)}{\partial x_j}=\phi(x'\beta)\beta_j\), где \(\phi(z)=\frac{1}{\sqrt{2\pi}}\exp\{-z^2/2\}\)

1.2 для logit

Рассмотрим logit-модель \(P(y=1)=\Lambda(x'\beta)\), где \(x'\beta=\beta_0+\beta_1x_1+\dots+\beta_kx_k\)

Предельные значения \(\frac{\partial P(y=1)}{\partial x_j}=\lambda(x'\beta)\beta_j\), где \(\lambda(z)=\frac{\exp(z)}{(1+\exp(z))^2}\)

1.3 Средние предельные значения

Обычно рассчитываются

  • предельные значения для каждого регрессора в средней точке (MEM - Marginal Effects at the Mean):
    • \(\phi(\bar{x}'\beta)\beta_j\) для probit
    • \(\lambda(\bar{x}'\beta)\beta_j\) для logit
  • среднее по всей выборке предельное значения для каждого регрессора (AME - Average Marginal Effects):
    • \(\overline{\phi(x'\beta)\beta_j}\) для probit
    • \(\overline{\lambda(x'\beta)\beta_j}\) для logit
